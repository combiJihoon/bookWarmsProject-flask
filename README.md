### 주요 기능
1. 독후감 작성 및 공유
2. 독서토론 스터디 생성 및 참여 기능
3. 데이터 분석 : 연령대, 성별, 취향별 인기책 표시 및 추천 
4. 헌책방 : 헌책방 위치 확인 & 한정판 또는 싸인책 구비여부 확인
5. 중고책 거래
